There's 2 ways to add javascript to your page.
  1. internally
    • adding a script tag inside the body
  2. externally
    • importing an extarnal script with the .js file type in the head
The console logs will show the extarnal script first, then the internal script.
This is because of the order the page code is read.
I don't suppose you need to add ; at the end of each line for the internal script
