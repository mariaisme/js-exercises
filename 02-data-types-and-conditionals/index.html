JavaScript String Methods

  • .length - it counts how many characters there are in the attached string
  • .charAt(n) - where n is a number, it takes the character in that position. Counting starts at 0
  • .charCodeAt(n) - where n is a number, it shows the UTF-16 code of the character at that position
  • .at(n) - similar to .charAt, but can handle negative indexes. If negative, it counts right to left instead of left to right.
Note:
if n is larger than the length of the string, it will be undefined

Extracting Parts of a String

  • slice(start, end)
    • starts at the first index, ends before the next index. Also accepts negative indexes.
  • substring(start, end)
    • similar to slice, but negative indexes are considered as 0.
  • substr(start, length)
    • starts at the index and ends at the specified length.
    • if length is not included, it will show the rest of the string
    • only accepts one negative index

String to Uppercase and Lowercase

  • .toUpperCase() - when attached to a string, it turns all of the characters to uppercase
  • .toLowerCase() - when attached to a string, it turns all of the characters to lowercase

Trim

  • .trim() - removes white spaces at the start and end of a string
  • .trimStart() - removes white spaces at the start of a string
  • .trimEnd() - removes white spaces at the end of a string

Padding

  • .padStart(length, padding) - pads a string at the start with the padding until the length is reached
  • .padEnd(length, padding) - pads a string at the end with the padding until the length is reached
  • .trimEnd() - removes white spaces at the end of a string
Note:
To pad a number, turn them into a string first.

Repeat

  • .repeat(n) - repeats the string for n amount of times

Replace

  • .replace(OldWord, NewWord) - finds the old word with the new word. Only works for the first word found.
  • .replaceAll(OldWord,NewWord) - replaces all the OldWord with the NewWord.
Notes:
This is case sensitive.
To find it regardless turn it into this /OldWord/i (insensitive)
To change all matches turn it into this /OldWord/g (global match).
To do both, turn it into this /OldWord/ig

String to Array

  • .split(",") - splits the string between the listed character ",". Only using "" will result in every letter being split
